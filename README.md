# DevSecOps - Lab 2

This repo contains a simple calculator app that is not so simple. It just looks simple. 

Created in python using uv package manager.

# Usage

- Clone and cd : `git clone https://github.com/Vedabahu/DevSecOps-thing; cd DevSecOps-thing`
- Use UV to run: `uv run main.py` 

# Testing

- All tests: `uv run python -m unittest discover tests`
- Individual tests: `uv run python -m unittest tests/[test_file.py]`
